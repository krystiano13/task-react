# React Recruitment Task

## Tech Stack I used:

- React
- React-Query
- React-Router
- TailwindCSS
- DaisyUI
- TailwindCSS-animated
- Typescript
- js-cookie

## Setup

- make sure you have Node.js instalaled on your machine
- clone repo
- run `npm i` command in project folder
- run `npm run dev` command in project folder
